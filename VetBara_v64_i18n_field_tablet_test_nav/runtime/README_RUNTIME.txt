VetBara runtime folder
======================

Node.js runtime target version: 20.18.1.

Fully bundled Linux x64 runtime is included when available in this build environment.

For macOS/Windows offline operation, place official Node.js binaries here:
- runtime/macos-arm64/bin/node or runtime/macos-arm64/node
- runtime/macos-x64/bin/node or runtime/macos-x64/node
- runtime/win-x64/node.exe

The bootstrap installer package can download and install these runtimes automatically from nodejs.org before the exam day.
